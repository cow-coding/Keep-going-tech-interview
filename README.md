**면접 질문과 개인적인 답변 내용들은 꾸준히 업데이트 됩니다.**

## 개인적인 작성 Rule

- 매일 각 분야별 질문 2개씩 선정해서 답변달기
- CS / AI 분야 번갈아가면서 답변달기
- 답변은 answer 폴더의 각 카테고리 문서에 작성
  - 너무 길어지는 것을 막고자 `날짜.md` 방식으로 관련 답변을 모음
  - 질문 리스트 문서는 answer에 있는 답변 문서로 링크 걸기
